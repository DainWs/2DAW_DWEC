## Array functions examples
### Slice
Slice devuelve un rango del array sin borrarlo de este.
> array.slice(1, 2) <br/>
> array.slice(posicion, posicionFinal) <br/>

#### Example
```js
array = [0, 1, 2, 3, 4, 5];
a = array.slice(2, 4);

a = [2, 3, 4];
```
